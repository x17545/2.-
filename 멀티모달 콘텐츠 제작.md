기획

브랜드 아이덴티티 : 인기가 증가할 것 같은 게임을 만들자

타겟 : 10\~30대

톤앤매너 : 추운 겨울/눈사람/전사

캠페인 목표/핵심 메시지 : 누구나 재미있게 게임을 즐겨줬으면 한다.

사용 도구와 목적 : OpeaAI Sora 2를 사용했고 동영상을 만드는데 비용이 유료이지만 다른 모델하고는 비용이 저렴했기 때문에 사용했다.



텍스트/이미지/영상 AI 장단점

                          

|  | 장점 | 단점 |
| :---- | :---- | :---- |
| 텍스트 AI | 사용자가 궁금한것을 알려주거나 규칙을 추가하여 여러가지 것을 만들어준다. | 텍스트로만 해서 눈이 재미가없다. |
| 이미지 AI | 눈이 재미가있어지고 여러가지 AI만의 이미지를 만들수있다. | 음악이 없어서 귀가 심심하다. |
| 영상 AI | 눈과 귀가 재밌고 독특한 영상도 제작할수있다. | 마감이 이상할수있고 불편할수도 있다. 영상이다보니 생성하는데에 있어서 시간이 많으 소모된다. |
